# &#x20;**Ultrasonic Sensor-Based Automatic Lid Control System**

# 

### &#x20;Overview



This project uses an HC-SR04 ultrasonic sensor and an SG90 servo motor to create a touchless automatic lid system.



### &#x20;Components Used



\* Arduino Uno

\* HC-SR04 Ultrasonic Sensor

\* SG90 Servo Motor

\* Breadboard

\* Jumper Wires



### &#x20;Working Principle



\* The ultrasonic sensor continuously measures distance.

\* When an object comes within 20 cm, the servo opens the lid.

\* The lid remains open while the object is detected.

\* After the object moves away, the lid closes automatically.
